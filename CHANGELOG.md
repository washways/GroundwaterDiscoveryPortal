# Changelog

All notable changes to the Groundwater Discovery Portal will be documented in this file.

## [1.0.0] — 2026-03-03

### Added
- Initial public release of the Groundwater Discovery Portal
- Three-pillar groundwater potential index (Storage, Supply, Yield) with multiplicative terrain constraint
- Interactive map with click-to-query diagnostics
- Auto-stretch visualization (local percentile scaling)
- Zoom guardrails to prevent oversized AOI failures
- Complete "Sources & Components" reference panel with per-dataset detail modals
- Detailed methodology modal
- National Malawi export function (90 m GEE Asset)
- Comprehensive README and supporting documentation
- Landing page for washways.org/groundwaterdiscoveryportal
